# Stock Data Explorer

A simple Python tool to download and visualize stock market data.

## Features
- Download historical stock data for any ticker
- Calculate key statistics (yearly change, high/low, volatility)
- Generate price charts
- Save visualizations as PNG files

## Requirements
- Python 3.7+
- yfinance
- pandas
- matplotlib

## Installation
```bash
pip install yfinance pandas matplotlib
```

## Usage

Run the script:
```bash
python stock_explorer.py
```

Enter a stock ticker when prompted (e.g., AAPL, GOOGL, MSFT, TSLA).

The script will:
1. Download 2023 stock data
2. Display key statistics
3. Generate and save a price chart
